# Changelog

## v0.1.0

Initial public release of BookSkill Studio.

Highlights:

- CLI for demo, run, validate, install, update, doctor, and open-report
- built-in demo source
- deterministic compiler and validator
- evidence-backed chapter, concept, and rules generation
- HTML, Markdown, and JSON validation reports
- local install support for Codex, Claude, Cursor, and generic agent homes
- GitHub Actions CI
